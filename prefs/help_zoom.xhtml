از منوی «پلاگین → بزرگ‌نمایی» می‌توانید اندازهٔ قلم و اجزای رابط زوترو را تغییر دهید. این قابلیت برای نمایش روی مانیتورهای کوچک یا ارائه‌های کلاسی مفید است. Use “Plugin → Zoom” to change the font size and interface scale. It is handy for small displays or classroom presentations. گزینهٔ «بزرگ‌نمایی (+)» نمای برنامه را یک گام بزرگ‌تر می‌کند. “Zoom In (+)” increases the interface size one step. گزینهٔ «کوچک‌نمایی (-)» برای بازگشت به اندازه‌های کوچکتر استفاده می‌شود. “Zoom Out (-)” steps the scale back down. گزینهٔ «حالت پیش‌فرض (0)» تنظیم را به مقدار استاندارد زوترو برمی‌گرداند. “Reset (0)” restores Zotero’s default zoom level. افزایش یا کاهش مقیاس در تنظیمات زوترو ذخیره می‌شود؛ بنابراین بعد از باز و بسته کردن برنامه نیز همان اندازهٔ خواناتر در دسترس خواهد بود. The chosen scale is saved in Zotero’s preferences, so the clearer size remains after restarting.
