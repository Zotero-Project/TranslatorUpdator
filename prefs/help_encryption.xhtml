قابلیت «قفل پیوست» امکان می‌دهد دسترسی به فایل‌های حساس را با گذرواژه محدود کنید تا هنگام باز کردن یا اشتراک‌گذاری، امنیت بیشتری داشته باشند. The “Attachment Lock” feature lets you protect sensitive files with a password before they can be opened or shared. روی فایل پیوست راست‌کلیک کرده و گزینهٔ «قفل پیوست» را انتخاب کنید. اگر گذرواژه‌ای تعریف نشده باشد، پنجرهٔ تعیین رمز باز می‌شود. Right-click an attachment and choose “Lock Attachment”. If no password is set, the password dialog appears. گذرواژه‌ای با طول ۶ تا ۲۲ کاراکتر وارد کنید و دوباره تکرار کنید. پس از تأیید، فایل در حالت قفل قرار می‌گیرد و در نمای آیتم علامت قفل نمایش داده می‌شود. Enter a password 6–22 characters long, confirm it, and the file will be locked. Zotero marks the attachment with a padlock badge. برای باز کردن فایل قفل‌شده، دوباره روی آن کلیک کنید؛ افزونه گذرواژه را می‌پرسد. بدون رمز صحیح فایل باز یا صادر نمی‌شود. To access a protected file, open it as usual; the plugin prompts for the password. Without the correct code the file cannot be opened or exported. برای برداشتن قفل، روی فایل راست‌کلیک کرده و «حذف قفل» را بزنید؛ پس از وارد کردن گذرواژهٔ فعلی، وضعیت به حالت عادی برمی‌گردد. To remove protection, right-click and pick “Unlock Attachment”; provide the current password to restore normal access. گذرواژه‌ها فقط روی دستگاه ظاهر خودتان ذخیره می‌شوند و در همگام‌سازی زوترو ارسال نمی‌گردند؛ بنابراین در صورت فراموشی رمز باید فایل اصلی را جایگزین کنید. Passwords are saved locally and never synced through Zotero, so keep a safe record. Forgotten codes require replacing the original attachment. برای تغییر گذرواژه ابتدا قفل فعلی را بردارید و سپس دوباره گزینهٔ قفل را اجرا کنید تا رمز جدید تعریف شود. To change the password, unlock the file first and then lock it again with the new credentials.
