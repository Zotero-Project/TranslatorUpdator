منوی «پلاگین» اکنون بخشی برای مدیریت افزونه دارد تا بدون خروج از زوترو بتوانید بستهٔ جدید نصب کنید یا افزونه را کامل حذف نمایید. The plugin menu now includes a management section so you can install updates or remove the add-on without leaving Zotero. «آپدیت پلاگین با فایل» یک بستهٔ .xpi یا .zip را انتخاب کرده و همان‌جا نصب/به‌روزرسانی می‌کند. “Update Plugin from File” lets you pick an .xpi or .zip package and install it immediately. «حذف پلاگین» افزونه را پس از تأیید کاربر پاک می‌کند تا در صورت بروز مشکل بتوان سریع برگردید. “Remove Plugin” uninstalls the add-on after confirmation, making recovery from problems quick. هدف این بخش کاهش نیاز به رفتن به فولدر پروفایل یا ابزارهای خارجی برای نگهداشت افزونه است و امکان پشتیبانی ساده در کلاس یا کتابخانه را فراهم می‌کند. This panel removes the need to dig into profile folders or external tools, streamlining support in classrooms and libraries.
