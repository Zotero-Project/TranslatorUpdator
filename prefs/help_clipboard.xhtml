با این قابلیت، امکان انتقال فایل ها از ویندوز به زوترو با استفاده از کلید های میانبر ویندوز فراهم شده است. Use these shortcuts to move files from Windows directly into Zotero without leaving the keyboard. crtl + V : انتقال فایل انتخاب شده از ویندوز به محل مورد نظر در زوترو Ctrl + V: Paste the copied file from Windows into the selected Zotero location. ctrl + B : عملکرد آن مانند ctrl + z است، یعنی فایل منتقل شده را به مبدا آن در ویندوز باز میگرداند. Ctrl + B: Undo the operation by restoring the transferred file back to its original Windows folder. عملکرد این قابلیت دوگانه است؛ در قسمت انتقال از کلیپ برد در بخش پلاگین، میتوان یکی از این دو حالت را انتخاب کرد، حالت همیشه انتقال که حالتی مخصوص است، اگر در ویندوز فایلی کپی یا کات شود، هنگام پیست داخل زوترو فایل به زوترو انتقال داده شده و از ویندوز حذف میشود. حالت دوم که حالت عادی است، کپی و کات هرکدام به ترتیب فایل را داخل زوترو کپی کرده یا انتقال میدهند. Two clipboard modes are available from the plugin menu. “Always import” removes the file from Windows after pasting it into Zotero, while the default mode follows the original action—copy keeps the file and cut moves it.
