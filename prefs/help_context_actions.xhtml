افزونه چند گزینهٔ جدید به منوی راست‌کلیک آیتم‌ها اضافه می‌کند تا بتوانید فایل‌ها را در تب یا پنجرهٔ مستقل باز کنید. The plugin adds extra right-click options so you can open attachments in their own tab or window. روی هر آیتم (کتاب، مقاله یا فایل پیوست) راست‌کلیک کنید تا منوی جدید نمایش داده شود. Right-click any item (book, article, or attachment) to reveal the enhanced menu. گزینهٔ «نمایش در تب جدید» آیتم یا فایل انتخابی را داخل تب جداگانه در محیط زوترو باز می‌کند. Choose “Open in New Tab” to display the selected item or file inside a dedicated Zotero tab. گزینهٔ «نمایش در پنجره جدید» همان محتوا را در پنجرهٔ شناور و مستقل باز می‌کند؛ مناسب برای مقایسهٔ چند مدرک. “Open in New Window” launches the same content in a floating Zotero window—ideal for side-by-side comparison. اگر آیتم فایل PDF داشته باشد، گزینهٔ «باز کردن PDF در تب جدید» فعال می‌شود و فایل را مستقیم در تب خوانش زوترو باز می‌کند. When a PDF is available, “Open PDF in New Tab” appears and routes the document straight to Zotero’s reader. در صورت نبودن فایل مرتبط، گزینه‌ها به‌صورت خودکار پنهان می‌شوند تا منوی اصلی شلوغ نشود. If an attachment is missing, the plugin hides the shortcuts automatically to keep the menu clean. برای شخصی‌سازی بیشتر می‌توانید در منوی «پلاگین → میانبرهای راست‌کلیک» ترجیح پیش‌فرض باز کردن در تب یا پنجره را انتخاب کنید. Open the plugin menu → “Clipboard Mode / Context Menu” section to switch the default behaviour between tabs and windows.
